Quran API Documentation

Your source for accessing the Quranic resources

Base URL

The base URL for the API is: https://quranapiuz.pythonanywhere.com/api/v3/quran

Endpoints

Get Sura Details

GET /<lang>/<sura_id>/

Retrieve the details of a specific Sura.

  • lang: Language of the Sura (e.g., uzbek, english)
  • sura_id: ID of the Sura

Example: GET /uzbek/1/

Get Ayat Details

GET /<lang>/<sura_id>/<oyat_number>/

Retrieve a specific Ayat from a Sura.

  • lang: Language of the Ayat (e.g., uzbek, english)
  • sura_id: ID of the Sura
  • oyat_number: Number of the Ayat

Example: GET /uzbek/1/1/

Create a New Ayat

POST /oyat/

Create a new Ayat.

Request Body: JSON object with the following fields:

  • sura_id: ID of the Sura
  • lang: Language of the Ayat
  • oyat_number: Number of the Ayat
  • translate: Translation of the Ayat
  • audio: URL to the audio file
  • author_id: ID of the author

Example Request:


{
    "sura_id": 1,
    "lang": "uzbek",
    "oyat_number": 1,
    "translate": "Tarjimasi",
    "audio": "https://example.com/audio.mp3",
    "author_id": 1
}
